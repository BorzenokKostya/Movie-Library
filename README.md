# Личная кинотека (Movie Library)

Это графическое приложение для хранения информации о фильмах.

## Возможности
- Добавление фильма (название, жанр, год, рейтинг)
- Сохранение данных в JSON файл
- Загрузка данных из JSON файла
- Фильтрация по жанру и году

## Как запустить
1. Убедитесь, что у вас установлен Python 3.x.
2. Склонируйте или скачайте проект.
3. В командной строке перейдите в папку с проектом.
4. Запустите команду:
```bash
python movie_library.py
